# Security Policy

## Reporting a vulnerability

If you believe you have found a security vulnerability in this repository:

- Prefer GitHub's **private vulnerability reporting** (Security tab) if available.
- Otherwise, contact the maintainer via GitHub: https://github.com/sammyjoyce

If the issue is not sensitive (for example, a build error or packaging bug), please open a normal GitHub issue.

## Scope

This repo contains Nix expressions and metadata to fetch and package AMD ROCm nightly tarballs.
It does not ship a network service.
